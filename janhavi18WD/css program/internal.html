Text Formatting

1. mark is used to highlight a word.

2. small is used for smaller text.

3. del is used to define deleted text, which usually strikes a line.

4. ins is used to insert text by underlining.

5. Formula of water is H2O. (subscript)

6. 25 is 52. (superscript)

7. em is also used for italic text.
